# New File Management System

This repository automates the provisioning and management of:
 - **Active Directory (AD) Domain Local security groups**
 - **NTFS and SMB permissions across file servers**
 - **DFS Namespaces and DFS Replication**
 - **Domain Local Group membership reconciliation using the CSV source of truth**
 - **NTFS child-permission normalization**
 - **DFS Namespace folder-target referral priority**

It uses a combination of:
- **Input files (Excel/CSV)**
- **Python-based CSV generator**
- **PowerShell submodules**
- **A main menu-driven orchestrator (`main.ps1`)**

## 💻 How to run

Step 1: **Navigate to the repository root directory**
```powershell
cd C:\Users\<your-name>\Documents\new-file-management-system
```

Step 2: **Launch the Main Orchestrator Menu**
```powershell
.\main.ps1
```

```powershell
=============================================
   FILE MANAGEMENT SYSTEM - MAIN MENU
=============================================
1. Generate CSV File
2. Create AD Domain Local Groups
3. Apply NTFS Permissions
4. Apply SMB Share Permissions
5. Configure DFS Namespace and Replication
6. Reconcile Domain Local Group Members
7. Normalize NTFS Child Permissions
8. Set DFS Referral target priority
9. Exit
=============================================
Select an option (1-7):

```
Each option executes a corresponding task:

| Menu Option | Action | Execution Context |
|---|---|---|
| **1** | Generates CSV files using the Python generator | Local client |
| **2** | Creates AD Domain Local groups | Domain controller |
| **3** | Creates folders and applies NTFS permissions | File servers |
| **4** | Creates SMB shares and applies SMB permissions | File servers |
| **5** | Configures DFS Namespace and DFS Replication | DFS root server |
| **6** | Reconciles Domain Local Group members against the CSV source of truth | Domain controller |
| **7** | Normalizes inherited permissions on child files and folders | File servers |
| **8** | Sets DFS folder-target referral | DFS root server |
| **9** | Exits the program | — |
                                  |
```

## Notes
- The Excel input file should be updated first before generating new CSV files.
- Always review the generated CSV files before applying AD, NTFS, SMB, DFS or Domain Local Group reconciliation.
